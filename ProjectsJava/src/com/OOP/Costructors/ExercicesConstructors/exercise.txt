La classe necessita di due campi (variabili di istanza) con nome width e height di tipo . double

  Metodo denominato getWidth senza parametri , deve contenere return il valore del width campo.

  Metodo denominato getHeight senza parametri , deve contenere returnil valore del height campo.

  Se il parametro è inferiore a 0 è necessario impostare il width valore del campo su 0.

  Metodo denominato setHeight con un parametro di tipodouble , deve impostare il valore del height campo.
